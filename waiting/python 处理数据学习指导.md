#### python 处理数据学习指导
*斜体*：用于新术语、URL、电子邮件和地址、文件名与文件扩展名。

等宽字体：用于表明程序清单，以及在段落中引用得程序中的元素，如变量、函数名、数据库、数据类型、环境变量、语句、关键字等。

等宽粗体：用于表明命令，或者需要读者逐字输入的文本内容。

等宽斜体：用于表示需要使用用户提供的值或者由上下文决定的值来替代文本的内容。
